### 技术栈选择
- **语言**: Python 3.x
- **依赖**: 仅使用标准库（`sys` 处理退出，内置 `input` 处理输入）
- **运行方式**: 命令行脚本

### 架构设计
- **单文件脚本**：`calculator.py`，包含所有逻辑
- **函数划分**：
  - `get_number(prompt)`：处理输入验证，返回合法数字或抛出错误
  - `add(a, b)`：纯粹的执行加法
  - `main()`：主流程控制，循环执行直到用户退出
- **错误处理**：使用 `try/except` 捕获 `ValueError`，提示用户输入有效数字

### 关键设计决策
- 使用 `float` 类型支持小数运算
- 提供循环模式，用户可连续计算
- 输入 `q` 或 `quit` 退出程序
