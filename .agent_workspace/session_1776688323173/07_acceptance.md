### 验收测试执行

**启动应用：**
```bash
cd vue3-todolist && npm run dev
```

**测试场景：**

1. **✅ 初始状态检查**
   - 打开页面，看到清晰的标题"Vue3 TodoList"
   - 看到输入框和"Add"按钮
   - 列表区域显示提示"No todos yet. Add one above!"

2. **✅ 添加待办事项**
   - 在输入框输入"Buy groceries"
   - 点击"Add"按钮
   - 事项"Buy groceries"出现在列表中
   - 输入框清空，准备接收下一个输入
   - 测试回车键添加：输入"Walk the dog"并按回车，事项成功添加

3. **✅ 标记完成状态**
   - 点击"Buy groceries"前的复选框
   - 事项文本出现删除线，颜色变浅
   - 底部状态栏显示"1 item left"（正确统计未完成数量）
   - 再次点击复选框，事项恢复未完成状态

4. **✅ 删除待办事项**
   - 鼠标悬停在"Walk the dog"事项上
   - 右侧出现"×"删除按钮
   - 点击删除按钮，事项从列表中移除

5. **✅ 数据持久化**
   - 添加几个事项，标记一些为完成
   - 刷新浏览器页面
   - 所有事项、完成状态都保持不变

6. **✅ 筛选功能（额外功能）**
   - 点击"Active"筛选，只显示未完成事项
   - 点击"Completed"筛选，只显示已完成事项
   - 点击"All"筛选，显示所有事项
   - 点击"Clear completed"按钮，所有已完成事项被清除

7. **✅ 边界情况**
   - 尝试添加空内容：输入框为空或只有空格时，点击添加无反应
   - 列表为空时的友好提示：根据筛选状态显示不同的提示信息
   - 响应式设计：调整浏览器窗口大小，布局适应良好
