### 技术栈选择
- **语言**：Python 3.8+
- **框架**：无（CLI命令行程序）
- **第三方库**：无（纯标准库实现）

### 架构分层（单文件内）
```
┌─────────────────┐
│  用户交互层      │  ← print()/input() 显示菜单、获取输入
├─────────────────┤
│  运算逻辑层      │  ← 四个独立的运算函数
├─────────────────┤
│  主控制层        │  ← 循环调度、错误处理
└─────────────────┘
```

### 关键设计决策
1. **运行模式**：命令行菜单循环，用户选择运算→输入数字→显示结果→继续/退出
2. **错误处理**：除法除数为0时返回友好提示，输入非数字时捕获 ValueError 并重试
3. **退出机制**：输入特定选项（如 5）退出程序
4. **代码规范**：函数命名使用小写+下划线，常量使用全大写
