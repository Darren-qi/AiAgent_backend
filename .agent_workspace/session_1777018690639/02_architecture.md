### 技术栈选择
- **后端框架**: FastAPI（轻量异步，自带Swagger文档和文件下载支持）
- **数据源**: akshare（开源A股数据接口，免登录，支持指数和成分股实时行情）
- **Excel生成**: openpyxl（纯Python，支持格式化和多sheet）
- **运行方式**: 单文件Python脚本，通过`uvicorn main:app --reload`启动

### 架构分层
```
┌─────────────────────────────────┐
│         FastAPI Web层           │  GET /api/stock-data
│   Swagger文档 / 文件下载         │
├─────────────────────────────────┤
│         业务逻辑层               │  数据获取 + Excel组装
│   akshare数据采集 + 格式化       │
├─────────────────────────────────┤
│         数据层                  │  akshare实时API
│   (akshare / sina实时行情)       │
└─────────────────────────────────┘
```

### 数据库选型
**不需要数据库** — 实时行情数据无需持久化，每次请求实时拉取并生成Excel提供下载

### 关键设计决策
1. **Excel结构**: 两个Sheet — Sheet1为三大指数实时行情，Sheet2为沪深300成分股行情（含股票代码、名称、最新价、涨跌幅、成交量等）
2. **数据源选择akshare而非tushare**: akshare无需token，开箱即用，适合本地运行
3. **错误处理**: 单个指数/成分股获取失败不影响其他数据，返回部分结果
4. **文件处理**: 使用`tempfile`创建临时文件，响应完成后自动清理

### 代码规范
- 类型注解（Python 3.10+）
- 函数单一职责，命名清晰
- 异常捕获并返回友好错误信息
